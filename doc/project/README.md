# Project Overview

스크립트 기반 프로젝트. 다양한 기능을 독립적인 스크립트로 구현하고, 각 기능별 문서를 체계적으로 관리한다.

## How to Add a New Feature

1. `doc/project/<feature-name>/` 디렉토리 생성
2. 하위에 `design/`, `plan/`, `implement/`, `result/` 디렉토리 생성
3. 위 테이블에 feature 등록
4. design 문서 작성 후 구현 시작
